Simple Stack VM

Минималистичная стековая виртуальная машина на языке C. Проект демонстрирует работу интерпретатора с использованием таблицы диспетчеризации (dispatch table) и валидацией состояния стека.

Особенности
- **Стековая архитектура**: все операции выполняются над данными в стеке.
- **Безопасность**: проверка Stack Overflow и Stack Underflow перед каждой инструкцией.
- **Производительность**: использование массива указателей на функции.

Поддерживаемые команды
- `BC_PUSH, .arg=<val>` — положить число в стек.
- `BC_IADD`, `BC_ISUB`, `BC_IMUL`, `BC_IDIV` — арифметические операции.
- `BC_INEG` — отрицание.
- `BC_ICMP` — сравнение двух верхних элементов (результат: 0 или 1).
- `BC_SWAP` — поменять местами два верхних элемента стека.
- `BC_DUP` — Положить на стек дубликат элемента.
- `BC_IPRINT` — вывести верхний элемент стека в консоль.
- `BC_IREAD` — считать число из стандартного ввода и положить в стек.
- `BC_STOP` — завершить выполнение программы.

Сборка и запуск

Для сборки проекта используйте `make`:

```bash
make
```

Запуск интерпретатора:

```bash
./vm_app
```

Структура проекта
- `vm.h / vm.c` — определение системы команд и описаний инструкций.
- `stack.h / stack.c` — реализация динамического стека.
- `interpret.h / interpret.c` — ядро интерпретатора и обработчики команд.
- `main.c` — точка входа и пример программы на байт-коде.
